Version : 0.0.1

# Vision

ThreinnOS est un système d'exploitation moderne développé entièrement de zéro.

L'objectif n'est pas de copier Linux, Windows ou macOS, mais de proposer une expérience différente, cohérente et performante.

Le système est développé en priorité pour une configuration matérielle officielle afin d'offrir une stabilité maximale.

---

# Objectifs

- Noyau monolithique
- Code entièrement écrit pour ThreinnOS
- Performances élevées
- Interface graphique moderne
- Utilisation simple
- Développement à long terme
- Compatibilité gaming importante
- Écosystème logiciel complet

Chaque composant doit être documenté.

Chaque module doit être indépendant.

Les performances ne doivent jamais sacrifier la stabilité.
